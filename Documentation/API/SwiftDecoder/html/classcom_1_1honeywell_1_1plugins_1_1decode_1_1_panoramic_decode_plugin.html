Detailed Description

A SwiftPlugin used for ensuring that n number of barcodes are decoded before the results are returned to the business logic. This plugin mainly allows a "Batch Scan" kind of functionality. The n number which refers to batch count can be set/get using API setDecodeResultRequirement The application needs to register PanoramicDecodeResultListener call back using HSMDecoder::addResultListener API to have further action on received barcode batch data. Use HSMDecoder::registerPlugin and HSMDecoder::unRegisterPlugin API to register/unregister any plugin on SwiftDecoder.
