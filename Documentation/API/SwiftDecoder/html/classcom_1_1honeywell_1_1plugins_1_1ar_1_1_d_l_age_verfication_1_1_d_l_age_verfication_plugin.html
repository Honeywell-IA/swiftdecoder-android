Detailed Description

A SwiftPlugin used for verifying the age detected on PDF417 barcodes within the preview. This plugin mainly allows the user to set an age and verify the age on the PDF417 DL The overlays on the preview allow the touch functionality get the licesne information. The application can register DLResultListener call back using DLAgeVerification::addResultListener API to have further action on selected barcode. Use HSMDecoder::registerPlugin and HSMDecoder::unRegisterPlugin API to register/unregister any plugin on SwiftDecoder.
