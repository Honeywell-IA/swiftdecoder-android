Detailed Description

This is the decoder base plug-in. It is used to give a base set of features to all plug-ins that derive from it.
This allows us to easily add a base set of features to a group of plug-ins. This is abstract and can only be sub-classed. This does not handle any results, as it relies on its children to add functionality.
