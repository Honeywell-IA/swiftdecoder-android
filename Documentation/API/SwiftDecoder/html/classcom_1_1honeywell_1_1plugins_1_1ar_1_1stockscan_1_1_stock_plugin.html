Detailed Description

A SwiftPlugin used for rendering stock information on a rectangular box hovering slightly above the detected barcode within the preview. The stock information that needs to be rendered should be coming from the integrating application through API onStockResult exposed on this plugin. For the format on how the Stock info should be populated, refer the StockResult class description. The stock info overlays on the preview allows the touch functionality to select a specific overlay. The application can register ARPluginTouchListener call back using StockPlugin::setOnARPluginTouchListener API to have further action on selected Stock Info Overlay. Use HSMDecoder::registerPlugin and HSMDecoder::unRegisterPlugin API to register/unregister any plugin on SwiftDecoder.
