Detailed Description

A SwiftPlugin used for scanning a batch of barcode either predefined batch or batch count till the user intervention. Option one is with predefined batch limit it ensures that n number of bar codes are decoded before the results are returned to the business logic. Option two is to get the batch count would be the n number of barcodes till the user intervention using API getScannedBatchResult or getScannedBatchCount .
