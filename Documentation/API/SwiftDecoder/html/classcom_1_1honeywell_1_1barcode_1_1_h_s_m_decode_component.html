Detailed Description

An HSMDecodeComponent is a real-time camera preview frame layout that can be include in your own activity for greater control over the look and feel of the barcode scanning operation. This allows you to resize the camera preview as you see fit. This can be used as a replacement for the HSMCameraPreview activity that is launched by the HSMDecoder scanBarcode() method.
An example of where you may wish to use this is within a tabbed activity. Using an HSMDecodeComponent embedded within your own activity allows you to customize the look and feel of the barcode scanning operation.
