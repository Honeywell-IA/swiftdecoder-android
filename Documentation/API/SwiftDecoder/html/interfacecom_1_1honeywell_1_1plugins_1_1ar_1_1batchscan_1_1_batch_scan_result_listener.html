void com.honeywell.plugins.ar.batchscan.BatchScanResultListener.onBatchScanResult ( HSMDecodeResult[]  barcodeData,
Bitmap[]  barcodeImages 
)

This method is called when a decode event has occurred. This will be called when a barcode was successfully decode or when a decode attempt has been terminated.

Parameters
barcodeDataThis is an array of HSMDecodeResult objects that contains all the data associated with each barcode decoded. If the decode result failed, this will return null.
barcodeImagesThis is an array of Bitmap images for each returned barcode. Since barcodes could be decoded across multiple frames we cannot get all images from last barcode image.
Returns
void
