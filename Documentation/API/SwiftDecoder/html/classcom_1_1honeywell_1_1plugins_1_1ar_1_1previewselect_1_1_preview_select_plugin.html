Detailed Description

A SwiftPlugin used for rendering Augmented Reality Overlay over the detected barcodes within the preview. This plugin mainly allows a "Preview & Select" kind of functionality along with capability to fetch the Preview Overlay count or Result data The overlays on the preview allow the touch functionality to select one barcode. The application can register DecodeResultListener call back using HSMDecoder::addResultListener API to have further action on selected barcode. Use HSMDecoder::registerPlugin and HSMDecoder::unRegisterPlugin API to register/unregister any plugin on SwiftDecoder.
